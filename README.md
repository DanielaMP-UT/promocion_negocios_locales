# Promoción de Negocios Locales

## Descripción

Este proyecto consiste en una página web para promocionar negocios locales de la La Pesca, mostrando información como el nombre del negocio, descripción de sus productos o servicios, dirección, horario de atención, número de contacto y fotografías.

## Instalación

1. Clonar el repositorio.
2. Abrir el proyecto en Visual Studio Code.
3. Ejecutar el proyecto en un servidor local.

## Uso

La plataforma permite consultar negocios organizados por categorías y visualizar la información detallada de cada negocio.
